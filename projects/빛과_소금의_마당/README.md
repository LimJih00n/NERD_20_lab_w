# 빛과 소금의 마당

**매체**: 5분 SF 단편 애니메이션
**소재**: 한국 성과급 사태 풍자
**시작일**: 2026-05-17

## 로그라인

현재 한국을 SF 미래에 빗대어 표현. 각자의 탐욕을 적나라하게 보여줘 개개의 탐욕의 역겨움을 느끼게 하고, 지금의 논쟁들이 미래를 망가뜨리는 행위임을 느끼게 한다.

## 파일

| 파일 | 내용 |
|---|---|
| `00_meta.md` | 14층 commits, 시점·인물·기적 결정 |
| `01_treatment.md` | 1차 rough treatment (작가 원문) |
| `02_outline.md` | 26 비트 정밀화 outline |
| `03_scenario.md` | shot-level 시나리오 |

## 현재 상태

- ✅ Treatment 1차
- ✅ Outline 26 비트
- 🟡 Scenario shot-level — Acts 1~2 완료, Acts 3~5 진행 예정
- ⏳ 시점·인물·기적 final commit
- ⏳ Prose draft (작가 직접)

## 작업 메모

- 시점 default: 카메라 객관
- 인물: 익명 코러스 (비네트 콜라주)
- 기적 한 가지: 빛→결정 (기계화는 세계 디폴트)
- 결말 톤: 카타르시스 0
- 채널 원칙: 어떤 인물도 결백하지 않음

## 작업 분담

- ✏️ **작가**: 씨앗·voice·commit·cut·최종 prose
- 🤖 **AI**: 구조 정리·비트 정밀화·시각 shot 1차·자료조사·편집 보조
- 대사: AI rough → 작가 polish
