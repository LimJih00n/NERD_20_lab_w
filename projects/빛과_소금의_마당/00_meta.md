# 빛과 소금의 마당 — Meta

## 로그라인

현재 한국을 SF 미래에 빗대어 표현한다. 각자의 탐욕을 적나라·직접적으로 (문장적·외형적) 보여줘 개개의 탐욕의 역겨움을 느끼게 하고, 지금의 논쟁들이 미래를 망가뜨리는 행위임을 느끼게 한다.

## 14층 commit

### 0층 — 색채·재미·시장
- **미감**: 잔혹 + 디스토피아 + SF 우화
- **정서**: 역겨움 + 황당함
- **결말 톤**: 카타르시스 0
- **채널 원칙**: *어떤 인물도 결백하지 않음*
- **시청자**: 한국 직장인·시민. 사건의 *주변인이지만 당사자처럼 느끼는* 위치.
- **trade-off 패턴**: A 동시형 (재미·시장·색채 모두)
- **레퍼런스 묶음**: Black Mirror Fifteen Million Merits / 봉준호 마더 / 콘크리트 유토피아 / Memories(오토모)

### 1층 — 전제
"한 거대 함선이 빛으로 결정 만드는 기술을 발견하자, 행성의 인프라가 무너질 때까지 모두가 약탈하고, 옆의 더 큰 세력이 조용히 다가온다."

### 2층 — 주제 (후보, 작가 cut)
- 후보 a: *"탐욕은 자기 행성을 어둡게 한다"*
- 후보 b: *"약탈하는 동안 옆의 더 큰 약탈자가 다가온다"*

### 3층 — 세계
- **기적 한 가지**: 빛 → 순수한 결정 변환
- 결정 = 사람·함선의 *연료*
- 사람들 *기계화* (세계 디폴트, 별도 기적 아님)
- 함선들이 *기계 행성*에 결정을 공급
- 행성에 결정 안 들어가면 *불빛 꺼짐*

### 4층 — 인물 구조
- **코러스**: 정치인·언론인·노조·시민
- **침묵하는 중심**: 행성 자체 (말 안 함, 무게 가짐)
- **화자/시청자 surrogate**: 없음 (비네트 콜라주)

### 5층 — 큰 골격
우화 5장 (`../../02_layers.md` 패턴):
1. **발견** (함선 1 결정 생산)
2. **첫 균열** (다른 함선·언론 폭로)
3. **외부 확장** (시위·정치·노조 가세)
4. **붕괴** (약탈·행성 꺼짐)
5. **코다** (옆 함선/행성 다가옴·노인만 남음·옆 행성 밝게)

## 시점·인물·기적 — 잠정 / 작업 중 정련

| 자리 | 현재 잠정 |
|---|---|
| 화자 시점 | 카메라 객관 |
| 따라가는 한 인물 | 없음 — 코러스만 |
| 함선 1 결정권자 | 미등장 (대신 임원 코러스로) |
| 정치인·언론·노조 명명 | 익명 (현실 우화 톤) |
