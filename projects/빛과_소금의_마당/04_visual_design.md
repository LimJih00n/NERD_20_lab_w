## 1. 행성

- **데스스타식 인공 구체**. 표면 = 도시 거리 + 모듈러 패널. 적도 트렌치 / 그래비티 링.
- 표면 곳곳에 *수직 거대 동맥 끝* — 도시 한가운데 솟아있음 (송전탑 자리를 대체하는 *살아있는 송전탑*).
- 어두운 자리 / 밝은 자리 = 모듈 단위로 lit/unlit.
- 행성 단면 와이드 컷에서 **중심에 기계 심장 + 증기기관 투입 장치**가 보임.

### 결정 흐름
```
[함선 노즐] → [행성 표면 주입]
     ↓
[표면 관로] → [심장부 옆 호퍼에 결정 쌓임]
     ↓
[화부(10대)가 호퍼에서 화로로 부어줌] ← 증기기관식 투입 장치
     ↓
[기계 심장이 박동 — 결정 연소]
     ↓
[여러 노즐 (동맥)로 흰빛 맥동]
     ↓
[행성 표면 도시 / 가로등 / 창문 — 심장 박자에 맞춰 밝아짐]
```

### 결정 부족 = 행성 죽음
- 호퍼 비어감 → 부어주는 양 줄어듦 → 심장 박동 느려짐 → 동맥 빛 약해짐 → 도시 꺼짐.

## 3. 색

### 베이스 톤 (영상 전체 5분)
- **회색·검정·콘크리트.** 차가운.
- 결정 빛 = 차가운 흰 (LED·별빛 느낌).
- 가로등·노즐 맥동 = 흰빛.
- 함선 내부 조명 = 차가운 청회색.
- 증기기관의 김 = **흰 차가운**. 화염 = *결정 불꽃의 흰 차가운* (스팀펑크의 따뜻함 거부).
- **휴대폰 화면 = 차가운 청백색 빛**. 시민의 얼굴을 *밑에서 비춤*.

### 가슴 결정의 빛 — 밝기 + 색 (두 변수)

> **디자인 원칙**: 결정은 사람당 **1개**. 평생 1개. 추가 분배·약탈 받으면 *결정을 더 박는 게 아니라* **그 하나의 결정의 밝기·색이 변함**.

**밝기 축** = 보유량 / 연료 수준
- 가장 밝음 → 거의 안 보임 (꺼진 빛까지)

**색 축** = 작가 cut 대기. 추천 안 (선택지 아니라 *시작점 제안*):
> **차가운 푸른 흰 ← 차가운 흰 ← 흐릿한 흰 ← 누런 흰 ← 꺼진 누런빛**
> 부자일수록 *더 차갑게 푸르스름*. 가난할수록 *식어가는 누런빛*.
> 함의: *부자의 빛 = 차갑다 (인간미 없음). 가난한 자의 빛 = 식어간다*.
> 옆 행성(B26) *붉고 노랑*과 대조 — 우리는 *식어가는 차가운 빛*, 옆은 *살아있는 따뜻한 빛*. 다른 자원/다른 시스템.

### 결정 받음 = 빛이 *변함*
- 분배 받으면 → 결정 빛이 *더 밝아짐 + 더 푸르스름해짐*
- 약탈로 새 결정을 *자기 가슴 결정에 흡수* → 같은 변화
- 결정 빼앗기거나 식어가면 → *흐릿해지고 누렇게 식어감*

### 따뜻한 색이 들어가는 자리 (영상 전체 통틀어 두 번)
1. **B20 약탈 후 바닥의 피·천·유리** — 흰 차가운 영상 5분 중 거의 유일한 따뜻함. *폭력만이 따뜻하다*.
2. **B26 옆 행성** — 5분 영상 전체에서 *가장 따뜻한 컷*. 붉고 노랑, 살아있는 듯. 차가운/누런 빛 5분 끝에 오는 *온도 점프*가 풍자의 마지막 칼.

### 단계적 어두워짐
- B7: 가로등 정상, 심장 박동 약간 느려짐
- B15: 가로등 깜빡, 박동 눈에 띄게 느려짐
- B21: 가로등 일제히 꺼짐, 심장 마지막 박동
- B24~25: 행성 거의 검정. 한 가로등만 희미하게.

## 4. 사람

### 공통
- 사지·척추 기계. **얼굴은 살**. 가슴 중앙에 **결정 1개**.
- 결정 **빛의 밝기 + 색** = 계급/연료 표현.
- 살 얼굴이라 **표정**이 풍자의 무기 (욕망·공포·행복·무표정).

### 부의 차이 = 다층 시각 표현 (잠금)

**한 컷에서 시청자가 시각으로 흡수할 수 있게 *세 가지* 동시 박힘**:

1. **가슴 결정 빛** (밝기·색): *차가운 푸른 흰* → *차가운 흰* → *흐릿한 흰* → *누런 흰* → *꺼진 누런빛*
2. **기계 사지·척추** (질감·재질): *정밀·매끈·반짝이는 금속* → *평범* → *거친·낡은* → *녹슨·삐걱*
3. **의복** (재질·색): *고급 정장·반짝* → *평범 셔츠/재킷* → *작업복* → *낡고 더러운 옷*

→ 가난한 자 = *누런빛 + 녹슨 기계 + 낡은 작업복* (3중)
→ 부자 = *푸른 흰빛 + 매끈 반짝 기계 + 고급 정장* (3중)
→ B4 거리 컷·B5 함선 내부·B19 약탈 등에서 *모든 차이가 한 프레임에* 박힘

### B20 (약탈 클라이맥스)의 가슴 빛
- 약탈한 결정을 자기 가슴 결정에 *흡수*. **빛이 폭발적으로 밝아지고 더 차갑게 푸르스름해짐**.
- 살 얼굴에 **피·먼지·땀** 묻은 채 가슴 빛이 폭발.
- 입은 조용, 눈만 밝아짐.
- *살 얼굴 + 더러움 + 후광* = 영상 중 가장 그로테스크한 컷.

### 결정의 시각 동작 (참고)
- 결정 자체는 *손가락 마디만 한 빛 알갱이*. 모양 변함 없음.
- 분배 받음 = 받은 결정이 *손바닥에서 사라지고* 가슴 결정이 *밝아지고 색 변함*. 또는 받은 결정이 가슴 결정에 *흡수되어 사라짐*.
- 약탈 = 잡은 결정이 가슴에 *흡수되며* 빛 변화. (개수 늘어남 X)

### 광장 군중 디자인 메모 (B17~B20)
- **남녀노소 다양한 군** — *어떤 한 성별·연령·계층의 행동*으로 보이지 않게 분산.
- 구성: 청년·중년·노년 / 남성·여성 / **인부 (성인 일반 노동자)** ·자영업자·사무직·노조원·하청 노조원 / 정치인·언론은 단상·카메라 자리. **화부(10대)는 광장에 없음** — 호퍼 옆 묵묵히 그대로 (미래 세대 상징).
- **제외**: *학생·어린아이*. 광장 약탈 현장은 *성인 욕망의 자리*. 학생·아이는 *시청자가 죄책감 느낄 출구* 또는 *피해자 surrogate*가 될 수 있어 제거.
- **얼굴 다양성**: 외형 차이로 *조직 안 됨*을 시각화. 같은 동작 (휴대폰 듦·외침·결정 가슴 박음)이 *다양한 얼굴에서 반복*되어 *모두가 그 욕망에 들어감*을 박음.
- *지친 얼굴은 광장에 없음* (엔지니어 카테고리 제거).

## 5. 행성 심장부

### 기계 심장
- **해부학적 심장 구조를 기계로 번역.** 심실 4개 → 거대 기계 챔버 4개. 판막 = 거대 밸브. 동맥 = 거대 노즐들.
- 박동 = 흰빛 *맥동*. 박동마다 노즐들로 빛이 흘러 나감.
- 행성 중심에 위치. 와이드 단면 컷에서 보임.

### 여러 노즐 (동맥)
- 심장에서 행성 표면 도시들로 뻗어 올라감.
- 표면에서는 도시 한가운데 *거대 동맥 끝부분*이 솟아있음.
- 결정 빛이 *맥동하며* 흐름 → 가로등·창문이 심장 박자에 맞춰 밝아짐.

### 증기기관식 결정 투입 장치 (심장 옆, 별도)
- 거대 보일러·실린더·압력 게이지·밸브·호퍼.
- 강철·콘크리트 (놋쇠 아님). 김 = 흰 차가운. 화염 = 흰 차가운 결정 불꽃.
- 함선이 표면에 주입한 결정 → 관로 따라 호퍼에 쌓임 → **화부(10대)가 호퍼에서 화로(심장 입구)로 부어줌** → 심장이 박동.

### 화부 (火夫) — **10대 노동 아이. 미래 세대의 상징**
- **반-기계 10대 노동 아이가 매일 결정을 *삽으로 퍼서* 화로에 부어줌.**
- **인부와 다름**: 화부 = *심장부 챔버 전담*. *10대 잠금*. 작업복 차림.
- **묵묵히 일만 함**. 표정 없음. **광장에 안 감 — 약탈 가담 X**. 어른들의 시위·약탈 소음이 *멀리서 들릴 뿐*, 화부는 그저 호퍼 옆에 *그대로*.
- **메타포 의미 (작가 cut 2026-05-18)**:
  - **태어나지 않은 미래 사람들의 상징** — 아직 오지 않은 자들.
  - **아직 어린 미래 한국 사람들의 상징** — 지금의 어른들이 망치는 미래.
  - *어른들이 약탈로 행성을 부숨 = 미래 세대의 일자리·삶을 부숨*.
  - 행성 꺼짐 = *미래 사라짐*. 가장 큰 피해자는 화부.
  - **B25 마지막 — 화부 자체 사라짐**. *늙은 화부도 없음*. 챔버 완전히 비어있음. 영상이 *어디 갔는지 말하지 않음*. *미래 자체가 사라진 자리* — 가장 강한 풍자.
- **노조가 아님.** 조직 없음. 일자리일 뿐. 발언권 없음.
- **결백 거부 원칙과의 관계**: 화부는 능동적 가해자 아님 — 단 *시스템 안에 매여 있음*. 그들이 *그 자리에 묵묵히 있다*는 것 자체가 *어른들의 행위에 대비*. 시청자가 *어른들의 욕망이 누구를 묻는가*를 발견.

#### 화부 외형 — **다음 결정 대기**
- **연령**: 10대 노동 아이 (잠금)
- 작업복: 강철 앞치마+안전모 / 평상복 그대로 / 혼합
- 얼굴 보호: 가스마스크·고글 있음/없음 (살 얼굴 잠금과 충돌 여부 — 10대 얼굴이 보이는 게 메타포 박힘에 유리)
- 인원·동작: 컨베이어식 다수 / 대장간식 소수 (10대 아이 *여러 명*인가 *한 명*인가)
- **광장 약탈(B17~B20) 때**: **화부는 광장에 없음**. 호퍼 옆에 *그대로 묵묵히* 있음. 결정이 없으면 *멈춰 있을 뿐*.

### 인부 (일반 행성 노동자) — **화부와 별개 카테고리**
- **다양 연령 성인 노동자** (청년·중년·노년 섞임).
- 운반·정비·잡일 등 행성의 *일반 노동* 담당. 화부와 *다른 직군*.
- B4 거리 onlooker, B11 댓글 보는 손, B17 행렬, B19~B20 약탈 등에서 *시민·자영업자와 함께 광장 군중*에 포함.
- 가슴 결정 = *흐릿한 누런빛* (행성 평민 수준).
- 화부와 시각 구분: *얼굴이 성인 + 작업복은 다양 (반드시 강철 앞치마는 아님)*.

### 광장 군중 메모 (§4)와의 관계
- 일반 광장 군중 = *학생·어린아이 제외* (성인 욕망의 자리)
- **화부 = 광장에 없음**. 호퍼 옆 *묵묵히 그대로*. *미래 세대의 묵음 자리*
- **인부 = 성인 노동자** — 광장 군중에 *남녀노소 다양 카테고리 중 하나*로 포함
- 시청자: 광장(어른들의 욕망) ↔ 챔버(묵묵한 미래) *시각 대비*가 풍자의 칼

## 6. 노조 모델 (재정렬)

### 두 종류로 분리

**a. 함선 노조** (대기업 정직원 노조 — 현실: SK하이닉스 정직원 노조, 삼성 노조)
- **자기네 영업익 % 더 달라**. 분배 요구 아님. *밥그릇 싸움*.
- 위원장 = B16 코러스 분열의 화면 컷에서 *자기 임금 협상*. 그리고 **B17 광장 시위에도 있음** — 자기 임금 요구 수준에서 출발. 가슴 결정 밝은 채로.
- **B18~B20 약탈에도 *함께 휩쓸려 가담*** — *그 현장은 인간 탐욕의 자리. 위원장도 결백 X*. 자기 결정 챙김. 광장에서 가슴에 박음.
- **일반 조합원 다수도 약탈 가담** — 시민·인부 사이에 흩어짐 (깃발·완장 없음). *이미 받았는데도 더 받으러 옴*.
- **B25**: 위원장 등장 안 함 유지 — *판단 보류*가 아니라 *광장 가담 후 시야 밖*. 운명은 시민·노인 카테고리 (잔해 속).
- **임원과 한 패 아님** — 위원장 결정도 일반 직원 수준. 단 *광장에서는 시민과 같은 손*.
- **노동권 측면 + 영업익 기여** — 단 *작품은 노조 쟁의를 "정당"하다 평가하지 않음*. 결백 X 원칙 유지.
- **작가 갱신 2026-05-18**: 이전 *"위원장 나쁘게 그리지 않음"* 결정을 *결백 거부*로 변경.

**b. 행성 하청 노조** (현실: SK 하청 노조)
- ***"우리도 동일 임금"*** 외침. **자기 임금 요구 — 함선 노조와 같은 톤의 작은 버전**. *"차별 마라"* 같은 호소 톤 아님.
- 외침은 **또렷**하지만 *언론이 안 다룸* (선택적 보도의 자리).
- 가슴 결정 *흐릿한 누런빛* (함선 노조보다 어두움). 단 *외침은 또렷*.
- **광장 약탈(B17~B20)에 *개별적으로* 합류** — 시민·인부 사이에 *손팻말 없이 흩어짐*. 자기 결정 챙김 — 함선 노조 일부와 같은 자리.
- **약자 X. 결백 X**. *밥그릇 싸움 참여자 — 자원이 적을 뿐*.

### 광장 군중은 노조 아님
- **시민 (조직 안 됨)**. 선동당해서 분노.
- **휴대폰 화면 들고 행진**.
- 노조 깃발 없음. 깃발 자리는 *언론 카메라·정치인 단상*.
- 약탈의 *손* 자체는 이들.

### 풍자의 칼날
> 권혁 (고려대 노동대학원): *"노조의 사회적 연대 가치가 후순위로 밀리고 밥그릇 싸움."*
- 함선 노조 = 자기 임금만 (노동권은 정당하지만 연대 가치 잃음).
- 시민 = 선동당해서 *남의 싸움*에 휘말려 약탈.
- 노조는 *광장에 안 옴* — 자기들은 이미 챙겼으니까.

## 7. 엔지니어 — 별도 캐릭터 군

> **원칙**: 함선 1·2의 일원이지만 *별도 캐릭터 군*. 일반 노조원과 *얼굴로 약간 구분* (지친 얼굴 vs 평온 얼굴). 함선 노조의 임금 요구 시위에는 참여. 광장 약탈 행렬엔 안 감. **단 카메라가 *안 갔다*고 강조하지 않음 — 그냥 광장에 없을 뿐**.

### 시각적 정체
- 함선 1·2 직원 중 *결정 만든 사람*. 옷·기계 사지·빛 = 다른 직원과 동일.
- 차이는 **얼굴** — *지침·주름·피로*. 일반 노조원은 평온 얼굴. 그것만이 시각 표시.
- 의류 = 함선 1·2 직원 유니폼 (별도 디자인 X).

### 작품에서의 자리

> **톤 원칙**: 영웅 X. 동정 X. *약간의 조명*만. 카메라가 *그가 광장에 안 갔다*고 변호하지 않음.

- **B2·B5**: 결정 첫 성공. *15년 노력 → 감격 → 욕망 전환*. 함선 1·2의 *한 엔지니어* 자리.
- **B3**: 분배 줄. 엔지니어가 *지친 얼굴*로 일반 직원들 사이에 섞여 같은 결정 받음. 카메라가 *지친 표정 한둘*을 짧게 클로즈업 — 조명은 그게 전부.
- **B6**: 임원 비공개 추가 분배. **엔지니어·일반 직원 모두 못 받음**. 박탈감의 자리는 *직원과 임원 사이*.
- **B10 (함선 2 시위)**: **엔지니어도 노조 일원으로 시위 군중에 *섞여 있음***. 외침에 참여. 지친 얼굴 한둘이 평온한 노조원들 사이에 있음. *함선 노조의 자기 임금 요구는 엔지니어도 함께함*.
- **B14**: 15년 노력 회상. *약간의 조명만*. 자막 *"15년."* 사실 표기. 카타르시스 X. 음악·시간 안 끌기.
- **B17 (광장 행렬)**: 광장에 *일반 노조원 다수 + 위원장 + 시민 + 정치인 + 언론 + 하청 노조원 일부* 섞임. **엔지니어만 광장에 없음** — 단 카메라가 그것을 *말하지 않음*. 그냥 광장에 *지친 얼굴이 없음*. 시청자가 알아챌 사람만 알아챔.
- **B19**: 시민이 함선 1로 쏟아져 들어올 때 *그 자리에 엔지니어 한둘 우두커니*. *지친 얼굴. 자기 결정 가림*. 시민은 못 봄. (다른 일반 직원·노조원·위원장은 광장에서 약탈 가담)
- **B20**: 함선 1 안 *엔지니어* 가슴 클로즈업. *이미 빛난 결정을 손으로 가리거나 놓지 않으려 함*. *다른 종류의 결백 거부 — 광장 가담 X, 단 자기 결정 놓지 않음*.
- **B25**: 마지막 노인 중 *지친 얼굴의 함선 1 출신 노인 한 명*. 별도 소품·강조 없음. *그냥 거기 있음*.

### 결백하지 않음 유지
- 엔지니어도 *욕망의 전환*이 있음 (B2).
- 함선 노조 시위에 *자기 임금 요구로 참여* (B10) — 정당하지만 *그도 받는 쪽*.
- 광장 안 감 ≠ 결백. 자기 결정 *놓지 않음* (B20).
- 시청자가 엔지니어에 *동일시되어 위로 받지* 않게.

### 일반 노조원과의 구분 (시각으로)
- **얼굴**: 엔지니어 = 지친 (주름·피로). 일반 노조원 = 평온.
- **시위 (B10)**: 둘 다 참여. 자기 임금 요구.
- **광장 약탈 (B17~B20)**: 일반 노조원 *다수* + 위원장 참여. 엔지니어 없음.
- 둘 다 결백하지 않음. 단 *광장에서의 행위가 갈림*.

## 8. 일괄 분배의 시각화

### 풍자의 핵심
> *"모두에게 똑같이"가 진짜 공정이 아니다.*

### 시각 자리

- **B3 함선 1 분배**: 엔지니어와 비-엔지니어가 *같은 줄*에서 *같은 크기 결정*을 받음. 표정만 다름. **카메라가 그 표정 차이를 클로즈업**.
- **B6 비공개 추가 분배**: 임원만 *별도로* 더 받음. 엔지니어 못 받음. *불공정의 두 번째 층위* (임원 vs 엔지니어).
- **B13 정치인**: *"모두에게 분배"* = **일괄 분배의 극단적 확장**. 가치 안 만든 자(시민)에게도 똑같이. 풍자의 칼.
- **B19~20 약탈**: 모든 시민이 *같은 행동*으로 결정 잡고 박음. *각자의 기여 차이 없이*. 일괄 분배의 그로테스크한 거울상.

### 정답 없음
- 차등 분배도 *밥그릇 싸움* (함선 노조)이 됨.
- 일괄 분배는 *공정함의 위장*.
- 작품은 **결론을 안 줌**. 카타르시스 0. 시청자가 *어떻게 해야 하나*만 남음.

## 9. 언론의 선택적 보도

### 다루는 것 / 다루지 않는 것이 캐릭터를 만듦

**언론이 다루는 것:**
- 임원 비공개 분배 (B8~9)
- 박탈감·계급화·17년 치 연봉
- 시민 댓글 (*"국민과 나눠라"* 등)
- 정치인의 *국민배당금형* 정책
- 시위·약탈의 *시민 분노* 측면

**언론이 다루지 않는 것:**
- **엔지니어의 15년 노력** (B14는 *카메라 객관*으로만 보여줌. 영상 안 TV에는 안 나옴)
- **함선 노조의 실제 요구** (임금 협상의 구체 내용)
- **하청 노조의 외침** (B16에 잠깐 등장하지만 무시당함)
- **약탈 중 우두커니 선 엔지니어** (B19. 언론 카메라가 *시민 분노만 찍음*)

### 시각화 방법

- B14 (15년 노력) 직후에 **TV·홀로그램은 *다른* 헤드라인** — 엔지니어 노력은 화면에 없음.
- B17 광장 행진 때 **언론 카메라가 *시민의 분노만 찍음***. 엔지니어는 카메라 프레임 *밖*.
- B25 마지막 화면들도 *충격·표·정치인*만. 누구도 *그동안 누가 결정 만들었는지* 묻지 않음.

## 9-A. 타락 시각화 — 언론·유튜버·정치인

> **위계**: 시민·노조원 = *욕망에 휘말림* (피해자이자 가해자). 언론·유튜버·정치인 = *욕망을 만들고 키워 자기 이익 챙김* (가해자이자 시스템 운영자). 둘 다 결백하지 않지만 타락 정도 다름. **노조 위원장은 여기 포함 안 됨** (작가 결정).

### 핵심 모티프: **카메라 앞 / 카메라 뒤** 양면

공식 면(분노·정의·시민 동원) + 뒷면(즐김·돈·계산·웃음)을 *한 컷 또는 짧은 전환*으로 보여줌. 시청자가 *동시에* 봄.

### 언론대표·관계자 (B12)
- *"1면에 박아"* 직후 = 책상 위 *과자·맥주·발 올림*
- *클릭 수 그래프 상승* 보면서 *입꼬리 천천히 올라감*
- 박탈감 댓글 보면서 *희희낙락* — 자기가 만든 분노를 *수확*
- 살얼굴이 *비대·기름짐*

### 자극적 정치 유튜버 (B11 끝 + B16 + B25)

> **톤 잠금** (작가 의도, AI는 대사 생성 X): 한국 현실 자극적 정치 유튜버 (참교육·박살 화법). 자칭 *정의의 사도·진실 추구자*. *시민 박탈감을 키우면서 노조도 동시에 박살냄*. *노조의 실제 요구를 흐림 + 시청자 분노를 노조로 향하게 함*.

**대사·썸네일 텍스트 = 작가 solo 자리.** AI는 시각·동작·맥락만 깔고 voice는 작가가 채움.

**참고 톤 키워드** (작가 참고용, 잠금 아님):
참교육 / 박살 / 진실 / 충격 / 각성 / 끝장 / 여러분

**작가 대사 작성 자리:**
- B11 끝 — 자극적 썸네일 제목 3~4개
- B16 — 카메라 앞 격앙 발화
- B25 — 마지막에도 같은 톤의 발화 (참교육·박살·후원 호소)

**B11 끝 (시각·동작)**: 시민 댓글 캡쳐 → 자극적 썸네일로 영상화. 시민 박탈감을 콘텐츠 재료로.

**B16 (시각·동작)**: 카메라 앞 → 격앙·검지로 카메라 찌르듯·벌게진 얼굴·침 튀김. 카메라 끄면 → *후원 알림·셀카·돈·입꼬리*. 가짜 분노 / 진짜 미소.

**B25 (시각·동작)**: 살아남은 자 셋 중 하나. 죽어가는 행성을 배경으로도 *후원 그래프 상승*.

**비주얼**:
- 방 = 호화 + 더러움. 책상에 *사이다·과자·에너지 드링크*
- 벽 = *음모론적 게시판* — 인물 사진에 *빨간 화살표·X*
- 모니터들 = *물량 공세* — 여러 썸네일 동시 표시
- 굿즈 = 자기 캐릭터 티셔츠·머그컵
- 가슴 결정 = 차가운 흰빛 (이미 잘 챙김)

### 정치인 (B13)
- 단상에서 *분노·격앙·"모두에게!"*
- 단상 뒤·차 안: *비서와 농담. 표 수 통계. 비서가 결정 봉투를 넘김* (자기 측근 분배)
- 입꼬리 *천천히 올라감* — 군중 환호 들으며
- 단상의 결정 빛은 *방송 조명 가짜 후광*. 무대 뒤에선 평범

### 반-노조 정치인 (B16)
- 단상에서 *벌게진 얼굴·삿대질·격앙* — *"파렴치한 노조놈들"* 톤 [작가 대사 확장]
- **다음 컷 — 그의 집무실 책상**. 깨끗·평범. 그 옆에 *노란 봉투 하나*. **출처 안 보임. 누가 두고 갔는지 없음. 그저 거기 있음**
- 봉투 = *뇌물 암시*. 결정 봉투(B13 정치인)와 *다른 색* — 그러나 *같은 기능*
- 카메라 *짧게* 비춤 — 시청자가 *자기 발견*하게. 손가락질 X
- 가슴 결정 = *차가운 흰빛, 평범한 정치인 수준*
- 시민 박탈감 + 노조 박살 *동시* 자극 (유튜버와 같은 위치, 단 *권력자 버전*)

### 살아남은 자 셋 (B25)
- 화면 안 정치인·언론·정치 유튜버 *여전히 잘 살고 있음 + 여전히 즐기고 있음*
- 정치인 — *"표를 주십시오"* 외친 직후 *카메라 끄고 측근과 웃음*
- 언론 앵커 — *"역대 최악"* 외친 직후 *시청률 그래프 상승 보고 입꼬리*
- 정치 유튜버 — *극단적 발언* 직후 *후원 그래프 상승·셀카·돈*

### 시민과의 연결 (양비론 거부)
- B11~B12: 시민 댓글(진짜 박탈감) + 언론대표(즐기며 박음) = *공범 관계*. 시민 분노가 *재료*가 됨
- B17 광장: 시민이 *자기 손에 든 휴대폰*에 *언론·유튜버·정치인 영상 켜진 채* 행진. *선동의 도구를 자기가 들고 옴*
- B19~B20 약탈: *진짜 시민의 손*. 언론·정치인은 그 자리에 없음. **시민도 결백하지 않음**
- 시청자가 *그들을 욕하면서 자기 손에도 휴대폰이 있다*는 것 발견

## 10. 휴대폰 화면 — 선동의 매개체

### 영상 전체 모티프
- **언론·정치인은 화면 안에서 말함.** 직접 광장 안 옴 (또는 단상 위에서만).
- **시민은 휴대폰으로 그 영상을 봄.**
- 광장 행진할 때도 **손에 든 휴대폰에 같은 언론·정치인 영상이 켜져있음** — *시민이 자기 손에 든 선동의 원천을 끌고 행진함*.
- B17 와이드 컷: **수백 개 휴대폰 화면이 광장의 유일한 조명** (가로등 꺼져가는 중).
- B20 가슴에 결정 박을 때 *휴대폰을 한 손에 든 채* 박음. 또는 휴대폰 떨어뜨리고 박음.
- B25 노인들도 같은 휴대폰. 화면 안에서 *정치인이 "표를 주십시오"*, *언론이 "충격 보도"*, **함선 노조 위원장도 *자기 임금 협상 중*** — 살아남은 자 셋.

### 시청자에 대한 시선
- 휴대폰 모티프가 *너무 명시적*으로 *선동의 도구*임을 외치면 시청자가 *자기는 다르다*고 느낄 위험.
- 해결: 휴대폰을 *비난의 대상*으로 그리지 않음. 그저 *카메라가 반복적으로 보여줌*. 시청자가 *자기도 똑같이 휴대폰을 들고 있다*는 것을 *스스로 발견*하게.

## 11. 시민의 얼굴 변천 — 마더식 살얼굴 활용

### 한 *성인 인부*의 시간 축 (광장 카테고리)
- **B1~B6**: 무표정·기계적 반복 (자기 일).
- **B4**: 거리에서 화려한 함선 1 사람들 봄. 자기 가슴 본다.
- **B11 이후**: 휴대폰 보다가 **얼굴이 변함** (배아픔).
- **B15~B17**: 작업복 채로 *광장행*.
- **B19~B20**: 약탈하는 손이 *일하던 손*. 가슴에 박을 때 **진심으로 행복함** — 자기 분노가 진실이라고 믿음.

### 한 *화부 (10대)*의 시간 축 (챔버 카테고리, 별도)
- **B1~B14**: 호퍼 옆에서 *묵묵히 부어줌*. 표정 없음. *어른들의 일은 그를 비껴감*.
- **B15**: 호퍼 절반 비어감 → 더 빠르게 부어주려 함. 어린 손 떨림.
- **B16 챔버 공허**: 결정 없어 *멈춤*. 그저 호퍼 옆에 *그대로*. 멀리서 광장 외침이 들림.
- **B21**: 호퍼 바닥남. 화부 *그대로 서있음*. 부어줄 게 없음. 화로 옆에서 *마지막 박동을 봄*.
- **B22**: 옆 함선이 호퍼·심장에 노즐 박음. *화부 어디 있는지 영상이 안 봄*.
- **B25**: **화부 자체 없음 — 챔버 완전히 비어있음**. *미래가 늙어버린 것도 아니고, 그저 사라짐*. *영상이 어디 갔는지 말하지 않음*. 가장 강한 풍자 — *어른들이 미래까지 빨아냈다*.

→ 두 시간 축의 대비: *욕망의 평민* ↔ *묵음의 미래*. 평민은 *늙어 살아남음* (B25 거리 노인). 미래는 *사라짐* (B25 빈 챔버). 둘 다 *어른의 행위로 인해*.

이 시퀀스가 *피해자이자 가해자*라는 미묘한 자리를 시각으로 박음.

| B10 | **시민 화활 — 노조 비판 댓글 폭발**. 작은 박탈감(B6.9) + 언론 부풀림 흡수 → 인공 분노. *"그들만의 잔치"* 톤. 시민 톤 — 임원 수치 관심 X, *노조에 화살*. *"노조 만들어라"는 여기 X — 직원 자리(B6.5)* |
| B11 | **자극적 정치 유튜버 (분리)** — 댓글을 캡쳐해 *자극적 썸네일·제목으로 영상화*. **노조 박살 톤** ("파렴치한 노조 박살" 류). 시민 박탈감 + 노조 박살 동시 자극 |
| B12 | 언론 공론화 → **노조 때리기 + 박탈감·계급화 헤드라인** 확장. 엔지니어 노력·임원 비리·노조의 실제 요구는 안 다룸. **언론대표 타락 컷** — *"1면에 박아"* 직후 책상 뒷면 과자·발 올림·클릭 수 그래프·입꼬리 |
| B13 | 정치인 *"모두에게 분배"* → **국민배당금형 정책**. 일괄 분배의 극단. **정치인 타락 컷** — 단상 분노 → 단상 뒤 비서와 농담·표 수 통계·결정 봉투 |
| B14 | **15년 노력 — 약간의 조명만**. 자막 *"15년."* 사실 표기. 영웅·동정·카타르시스 X |
| B15 | 가로등 깜빡 → **호퍼 절반 비음** + **화부**가 더 빠르게 부어주려 하지만 결정 모자람 + 박동 눈에 띄게 느려짐 |
| B16 | 코러스 분열 → TV에서 함선 노조 위원장(자기 임금 협상, 타락 시각화 X) / 정치인(분배) / 언론(박탈감) / 시민·자영업자·노인 — 자기 화면 보면서 외침. **유튜버 컷 양면 확장** — 카메라 앞 (분노·진실 추구하는 척) + 카메라 끄면 (광고 수익·셀카·웃음). **반-노조 정치인 추가** — *"파렴치한 노조놈들"* 격앙 + 다음 컷 *집무실 책상 옆 노란 봉투 하나* (출처 안 보임. 시청자가 자기 발견). **행성 하청 노조 한 컷** — 외곽 거리, *"우리도 동일 임금"* 작은 손팻말 (자기 임금 요구, 함선 노조 작은 버전, 약자 X), 가슴 결정 흐릿하지만 외침 또렷, 언론 안 다룸. **노조 깃발 없음** (하청 손팻말 외) |
| B17 | 시위 격화 → 광장 = 시민 (조직 안 됨). **수많은 휴대폰 화면 든 손들**이 광장 조명. 정치인 단상·언론 카메라는 있음. **함선 노조 조직은 없음 — 단 *위원장 + 일반 노조원 다수*가 가슴 결정 밝은 채 시민·인부 사이에 흩어짐 (깃발 없음). 하청 노조원도 한둘 흐릿한 가슴으로 시민 사이에 흩어짐 (손팻말 없음)**. **엔지니어만 광장에 없음** — 단 카메라가 그것을 강조하지 않음 |
| B18 | 누군가 뛰어듦 → **언론 카메라 부서짐 / 정치인 단상 깃대 부러짐** (노조 깃발 X) |
| B19 | 결정 약탈 → 시민이 함선 1로. **함선 1 안 한쪽에 *엔지니어 한둘* 우두커니** (*지친 얼굴. 자기 결정 가림*). 시민은 못 봄. **시민 사이에 섞인 일반 노조원 다수·하청 노조원 일부 + *위원장*도 약탈 가담** (위원장 가슴 결정 이미 밝은 채로 결정 잡으러 손 뻗음) |
| B20 | 가슴에 박음 → 시민·인부·노인·자영업자·**일반 노조원 다수·하청 노조원 일부·*위원장*** 가슴 빛이 폭발적으로 밝아짐 (광장). 위원장 가슴 클로즈업 = *결백 X*. **정치인·언론은 자기 사무실·방송국에서 같은 동작**. **함선 1 안 *엔지니어* 가슴 클로즈업** — 자기 결정 빛을 손으로 가리거나 놓지 않으려 함 (*다른 종류의 결백 거부*) |
| B21 | 호퍼 *바닥남*. **화부는 그 자리에 그대로 서있음** — 부어줄 게 없음. 마지막 결정 부어지지 않고 떨어져 있음. **심장 마지막 박동**. 화부 *멀리서 들리는 광장 환호 안 봄. 그저 화로를 봄* |
| B22 | 옆 함선 노즐이 표면 뚫고 **호퍼·심장 둘 다 직접 박힘**. 남은 결정·잔여 빛 빨려나감 |
| B23 | 작은 함선들 떠남 (부유 지역 + 함선 1 임원). 가슴 결정 밝은 자들 → 옆 함선/옆 행성으로 합류 |
| B24 | 도시 점점 꺼짐. **광장 잔해 와이드** — 부러진 로봇 팔·뜯긴 척추·짓이겨진 살얼굴·부서진 가슴 결정·뒤틀린 몸·피·천·짓이겨진 휴대폰들 (지지직 정치인 영상 재생). **콜드 오픈 잔해와 동일 자리** — 시청자가 알아챔. 잔혹 수위 톤 (Cronenberg·박찬욱·Berserk) 유지 |
| B25 | **별도 짧은 컷 — 챔버 완전히 비어있음**. 호퍼·화로 텅. **화부 사라짐 — 영상이 어디 갔는지 말하지 않음**. 삽만 바닥에. 미래 자체 사라진 자리. // **거리 — 살아남은 시민 노인** 한둘 (인부 노인 + 지친 얼굴 함선 1 출신 노인 한 명). 가슴 결정 거의 꺼진 누런빛. 화면 안에서는 **정치인·언론·자극적 정치 유튜버 셋** *여전히 차가운 푸른빛 + 여전히 즐기고 있음* (카메라 앞/뒤 양면 짧은 컷) — **살아남은 자 셋**. 노조 위원장 영상 등장 안 함 (운명 보류) |
| B26 | 옆 행성 = 같은 데스스타 종, 심장부가 박동 (붉고 노랑) |
