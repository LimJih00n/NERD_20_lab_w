# 베르세르크 — 0~14층 분해

**작품**: 베르세르크 (Berserk)
**작가**: 미우라 켄타로 (1989~2021, 사후 모리코우지 스튜디오)
**매체**: 만화 (+1997·2012·2016 애니메이션 / 2012 영화 3부작)
**장르**: 다크 판타지, 액션, 호러

**분해 시점**: 2026-05-16 (writing_labs 시스템 시범 분해)
**분해 목적**:
1. 14층 알고리즘이 작동하는지 검증
2. 시스템에 박을 매크로 후보 발견
3. *재미 + 시장 + 색채* 셋 다 잡은 작품의 패턴 추출

## 0층 — 재미·시장·색채

### 재미 차원 (hook)

| 차원 | 베르세르크의 자리 |
|---|---|
| 주된 끌어당김 | *극단성*. 다른 만화에서 못 얻는 *진짜 무게* — 폭력·잔혹·성·트라우마·무력감이 우회 없이 직격 |
| 캐릭터 hook | 가츠의 *압도적 복수 동력* + 거대한 검 (Dragon Slayer) 한 자루의 시각적 상징성 |
| 관계 hook | **친밀했던 동지 → 가해자**의 구조 (가츠 ↔ 그리피스). 단순 영웅-악역 X. *사랑했던 자가 자기 욕망 위해 너를 제물로 바침* |
| 세계 hook | 중세 + 종교 + 마계가 *겹쳐 있는* 세계관 깊이 |
| 그림 자체 hook | 미우라의 디테일 — 표정·신체·환경의 정밀도. *읽는 행위 자체가 보상* |
| 카타르시스 거부 | 이긴 자도 잃음. 구원 없음. → *위안 구조*에 질린 독자가 *진짜 무게* 찾아옴 |

### 시장 차원

| 차원 | 자리 |
|---|---|
| 출발 시장 | 일본 청년 만화(seinen) 매니아층 — *컬트* |
| 확장 경로 | 1997 애니 → 2010년대 영화 3부작 → 2016/17 애니(악평) → **게임 산업 우회 유입** (다크 소울·엘든 링·디아블로) |
| 시장 위치 | *어른의 다크 판타지* 정전(canon). 영어권에서 *seinen manga 입문 후 도달하는 정상* |
| 시장 따라온 이유 | 미우라가 *시장 안 따라감*. 30년 자기 페이스 → *그 자리에서만 얻는 무게*가 시장에서 희소재 |
| 작가 사망(2021) 효과 | *컬트 → 정전* 격상. 미완성의 *불가역적 가치* + 추종자 결집 |

### 색채 차원

| 차원 | 자리 |
|---|---|
| 톤의 무게 | *무력감의 정점까지 박음*. Eclipse 한 시퀀스가 압축 |
| 고통의 신성화 거부 | 폭력은 *의미 없이* 잔혹. 의미는 *인물이 부여*. 작가가 해석 안 함. |
| 위선 폭로 | 종교(Conviction)·정치(매의 단 후반)·도덕에 대한 위선. *그리피스가 도덕 언어 안 씀 + 행동만으로 자기 신성화* |
| 친밀의 무기화 | 가장 친밀했던 사람이 *자기 욕망 위해* 너를 제물로. *적의 폭력보다 사랑한 자의 폭력이 더 무거움*. |
| 그림이 곧 색채 | 미우라 그림체 = 정체성의 절반. *재현 불가*. |
| 카타르시스 0 | 이겨도 잃음. 가츠는 *복수의 가능성*만으로 살아감 |

### Trade-off 룰

**미우라의 선택: 색채 > 재미 > 시장 (= B 색채 우선형)**

```
색채 100% 박음 → 재미는 그 색채의 부산물 → 시장은 30년 뒤 따라옴
```

| 패턴 | 작가 예 | 베르세르크 위치 |
|---|---|---|
| A. 동시형 | 봉준호·박찬욱·황동혁 | ❌ |
| **B. 색채 우선형** | **미우라·검은신화·박찬욱 초기** | ✅ |
| C. 시장 우선형 | 양산 만화·웹툰 | ❌ |

### 0층 안티템플릿 후보 (베르세르크에서 추출)

- ❌ 카타르시스 박기 — 이긴 후 보상 주는 디폴트
- ❌ 폭력 의미화 — *왜 이 폭력이 일어났는지* 작가가 설명
- ❌ 영웅 회복 보장 — 잃은 것이 결국 되돌아옴
- ❌ 적의 단순화 — *순수 악*의 적. 가해자가 카리스마·매력 없음
- ❌ 친밀의 안전성 — 친밀한 관계가 *안전 자산*이라는 디폴트

## 1층 — 전제

**채택**: "한 야망가가 자기를 사랑한 자들을 제물로 신이 되고, 살아남은 한 사람이 그를 죽이러 간다."

- *사건*: 야망가의 신성화 의식 + 추적의 여정
- *반전*: 신이 되는 대가가 *사랑한 사람들의 제물화*
- *변화*: 친밀의 무기화 → 살아남은 자의 추적 동력

핵심 어휘 두 개 — *제물*과 *살아남은 한 사람*.

## 2층 — 주제

**채택**: "가장 큰 꿈을 가진 자가 자기를 사랑한 사람들을 가장 먼저 베었다."

- 행위적: *베다* — 동사
- `04_macros.md` 예시 *"가장 많이 가진 자가 가장 먼저 베었다"*와 같은 호흡
- 미우라의 *야망 신성화*에 대한 정확한 입장
- *베다*가 작품의 모든 검·폭력과 직접 연결

대안 후보: *"꿈은 친밀했던 자를 제물로 만든다."* (더 압축, 그러나 *베다*의 구체성 약함)

## 3층 — 세계

### 시대·장소
- 중세 유럽 풍 — *정확한 라벨 없음*. 전쟁 일상화된 봉건 사회
- 미들랜드 왕국 + 주변 토지 + *겹쳐진 영적 영역(마계)*

### 핵심 기적 — **한 가지 묶음**

```
베헬릿 — 갓핸드 — 인과(Causality)
     ↓
인간이 가장 사랑하는 것을 제물로 바쳐
사도(Apostle) 또는 갓핸드의 일원이 된다.
운명은 미리 정해져 있다.
```

세 단어처럼 보이지만 *한 기적*. 다른 마법·괴물·종교는 이 기적의 변주.
→ `02_layers.md`의 *기적은 한 가지* 룰 정확히 지킴.

### 세계의 결정적 선택

| 선택 | 미우라의 답 |
|---|---|
| 마계는 인간이 만들었나? | ❌ 원래 있었음. 인간보다 큼. |
| 자유의지 있나? | ⚠️ 인과가 거의 모든 것을 결정. 자유는 *작은 틈*. |
| 신은 인간 편인가? | ❌ 신은 인간을 *이용* (갓핸드) |
| 종교는 진짜 구원인가? | ❌ 교회는 위선 (Conviction arc) |

### 시스템 학습 자리 — *시각 설정 최대 + 텍스트 설정 최소*

미우라는 *세계 디테일이 매우 많은* 작가. 그러나 `02_layers.md` *최소 설정 원칙* 위반 아님:
- *기적은 한 가지*만 박음
- 디테일은 *그림*에 박음 (텍스트 아님)
- 시스템의 본질(설명 안 하기)은 지킴

→ **사용자 채널 함의**: 영상에서 세계 디테일은 *시각으로 박고 텍스트로 안 박는다*.

## 4층 — 인물 구조

### 핵심 삼각 — 작품의 *원자*

```
        가츠
      (살아남은 자)
      야망 없음
         |
      ─────────
      |        |
   캐스카 ← → 그리피스
   (침묵)    (야망가)
   Eclipse  신이 된 자
   파괴됨
```

세 인물이 작품의 *원자력*. 모든 다른 관계가 여기서 파생.

### *침묵하는 중심* — 캐스카

`02_layers.md` *침묵하는 중심* 패턴 완벽 본보기.

- 캐스카가 작품의 *정서적 중심*
- Eclipse 후 *말하지 못함*
- 캐스카가 말하면 *그 자리가 약해짐*
- 침묵이 *Eclipse의 무게를 지속*시킴

→ **사용자 채널 함의**: 작품의 가장 약한 자, 가장 침묵하는 자가 *주제의 무게를 지님*.

### 인물 묶음

| 묶음 | 역할 |
|---|---|
| 매의 단 (옛) | 코러스. Eclipse에서 *제물*. 집단 죽음. |
| 새 동료 (Schierke, Farnese, Isidro 등) | 가츠의 *재구성된 가족* |
| 그리피스의 새 매의 단 | 옛 매의 단의 *유령적 재현* — 위선 깊이 더해짐 |
| 갓핸드 | *위에서 조작* — 직접 손대지 않음 |
| 적대 사도 | 각 arc 변주. *욕망 위해 사랑한 것 제물로 바친* 사례들 |

→ **반복 패턴**: 모든 묶음이 *한 인간이 자기 욕망을 따른 결과*의 변주. 4층 인물 구조가 2층 주제의 *반복 증명*.

## 5층 — 큰 골격

### 외적 골격 — Arc 구조

```
[프롤로그]      검은검사편 — *씨앗의 결과* 미리 보여줌
                                            ↓
[회상]          황금시대편 — *씨앗 사건* 자체 (Eclipse)
                                            ↓
[현재 여정]     Conviction → Millennium → Fantasia
                                            ↓
                                          (미완)
```

### 골격 패턴 — **씨앗-회상-여정**

3막도, 우화 5장도, 비네트도 아님. 미우라가 만든 골격:

1. **씨앗의 결과** (현재 시점)로 시작 — 독자가 가츠를 *이미 망가진 상태로* 만남
2. **씨앗 사건**(회상) — 어떻게 망가졌는지 천천히. 회상이 본편보다 길고 무거움.
3. **그 후 여정** — 회복인지 복수인지 정착인지 모호한 추적

→ 일반 *시간 순*으로 갔으면 — Eclipse가 *클라이맥스로 소비*됐을 것. 회상으로 박으니 *영원히 지속되는 그림자*.

### 내적 골격 — 가츠의 변화

1. 고통받는 아이 (회상 외)
2. 공동체 발견 (매의 단 합류)
3. 공동체 상실 (Eclipse) ← 작품 무게 중심
4. 추적 동력 (검은검사 시기)
5. 새 공동체 + 추적의 모호함
6. 마지막 (미완 — 영원히 미완)

### 함의

- 작품이 *닫히지 않음*. 씨앗(Eclipse)이 *영원한 현재*로 작동
- *카타르시스 0* (0층 색채와 정확히 묶임)
- 미완으로 끝나도 *작품 무게 손상 안 됨* — 본래 *닫힘이 없는* 구조
- 미우라 사망과 작품 구조가 *일치* (우연 아닐 수도)

## 6층 — 시퀀스 (Eclipse 시퀀스)

```
1. 도입 — 매의 단의 짧은 정착 (행복)
2. 베헬릿 활성화 — 모두 마계로 끌려감
3. 갓핸드 등장 — "제물을 바쳐라"
4. 그리피스의 선택 — 사랑한 자들을 가리킴
5. 매의 단 학살
6. 페모토 변신 + 캐스카 강간 + 가츠 손·눈 잃음 ⭐
7. 황혼의 들판 (탈출 후 정적)
```

**패턴: 에스컬레이션 시퀀스 (극단형)**

- 각 단계가 *이전보다 무겁게*
- 끝에서 *카타르시스 없음*
- 황혼 = 휴식이 아니라 *모든 것이 파괴된 후의 정적*

**한 변화**: *친밀의 무기화 완성 + 가츠·캐스카 정체성 파괴*

→ 5층 골격에서 *씨앗이 박히는 자리*가 정확히 여기.

## 7층 — 장면 (Eclipse 7개 장면)

| # | 시간 | 장소 | 변화 |
|---|---|---|---|
| 1 | 일출 직전 | 들판 (지구) | 행복 → 의식 시작 |
| 2 | 시간 없음 | 마계 | 갓핸드 요구 → 그리피스 선택 |
| 3 | 시간 없음 | 마계 들판 | 사도들이 매의 단 도살 |
| 4 | 시간 없음 | 마계 | 가츠·캐스카 도주 — 동료들 보면서 못 멈춤 |
| 5 | 의식 절정 | 갓핸드의 자리 | 그리피스 → 페모토 |
| **6** | **의식 정점** | **사도들의 극장** | **페모토가 캐스카를 가츠 앞에서 + 가츠 손·눈** ⭐ |
| 7 | 일몰 | 들판 (지구) | 두 사람 누움. 캐스카 정신 잃음. 가츠 영원히 변함. |

장면 6 = *시퀀스의 무게 중심*. 비트 분해는 여기서.

## 8층 — 비트 (장면 6 분해)

```
[6-1] 가츠가 페모토를 보고 *그가 그리피스였음을 알아챔*
      행동: 형태의 잔여에서 그리피스 식별
      변화: 적이 *낯선 존재 → 사랑한 사람*으로 전환
      유형: 인식 비트

[6-2] 사도가 가츠의 손을 입 안에 물음 — 가츠 *행동 불가능*
      행동: 사도가 물고만 있음 (씹지 않음). 가츠 발버둥치지만 못 빠져나옴.
      변화: 가츠가 *행동 주체 → 목격자*로 강제 전환
      유형: 결정 박탈 비트 (★ 새 매크로 후보)

[6-3] 페모토가 캐스카를 향해 다가감 (시간 느림)
      행동: 페모토의 손이 *느리게* 캐스카에 닿음
      변화: 의식의 의도가 *물리적으로* 시작
      유형: 시간 압축 비트 (분노 매크로의 *시간 느려짐*과 같은 메커니즘)

[6-4] 캐스카의 인식
      행동: 캐스카가 *페모토 = 그리피스* 깨닫는 순간 — 표정의 무너짐
      변화: 캐스카의 *정신적 정체성* 깨지기 시작
      유형: 인식 비트

[6-5] 페모토의 행위 (직격 + 암각 혼합)
      행동: 페모토의 입이 캐스카에 닿음 → 부분만 보임
              사도들의 머리가 둘러싸 *바라봄*
              가츠·캐스카의 *눈*이 핵심 이미지
      변화: 사건 일어남 — 그러나 직격으로 안 보임
      유형: ★ 새 매크로: *강제 목격*

[6-6] 가츠의 자기 절단
      행동: 가츠가 자기 손을 도끼로 끊어 빠져나옴
      변화: 자기 신체 절단으로 주체성 일부 회복 — 너무 늦음
      유형: ★ 새 매크로: *자기 절단으로 주체성 회복*

[6-7] 가츠의 무너짐 + 눈 잃음
      행동: 캐스카에게 손 뻗지만 너무 멀음 → 사도 발톱이 가츠 눈 파괴
      변화: 시각 일부 영원히 잃음 — *가장 중요한 것을 불완전하게 본 채* 종료
      유형: ★ 새 매크로: *불완전 목격*
```


★ 새 매크로 후보: **반응 부재** — 비명·울음·외침이 *있어야 할 자리에 0개*. 시청자가 *반응을 채움*.
