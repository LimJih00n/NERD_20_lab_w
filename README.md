# NERD_20 작업 코덱스 — 시나리오 에이전트 시스템

AI 협업 시나리오 작업 시스템. 단순 코덱스에서 *에이전트 시스템(A+D)*으로 진화 중.

목표: **재미 ∩ 잘 팔림 ∩ 색채** — 셋 다 잡는 시나리오를 자동 생성하는 시스템.

## 무엇인가

이 코덱스는 *매크로 라이브러리*가 아니라 **상상력 보철물**이다. LLM이 구조적으로 가지지 못한 14차원의 상상을 외부에서 공급하는 시스템. 시간이 갈수록 두꺼워지고, 두꺼워질수록 출력이 *사용자만의 작품*에 가까워진다.

매크로의 진짜 본질: **AI 상상 공간의 크기를 *확장 + 깊이* 두 방향으로 키우는 지도**. 자세히: `01_principles.md`, `04_macros.md`.

## 시스템의 진화 — *천장 깎기 주체 이동*

| 원래 코덱스 | 에이전트 시스템 |
|---|---|
| 사용자 = 제약 공간 좁힘 (천장 = 정밀도) | 사용자 = *판단·승인·거부* |
| AI = 좁은 공간에 채움 | **AI = 제약 공간 좁힘 자체** |
| 매크로·14층 = 참고 자료 | 매크로·14층 = 에이전트의 내장 craft |

자세히: `11_agent_system.md`.

## 어떻게 쓰는가

### 새 작품 시작 시 (에이전트 흐름)

```
[입력] 사용자 한 줄 (소재·각도)
   ↓
[0층 Producer]    재미·시장·색채 trade-off    ⚖️ GATE-0
   ↓
[1~5층 Director]  전제·주제·세계·인물·골격    ⚖️ GATE-1·2·3·4·5
   ↓
[6~8층 Beat Builder] 시퀀스·신·비트            ⚖️ GATE-6·7·8
   ↓
[9~11층 Cinematographer] 시점·컷·음향          ⚖️ GATE-9·10·11
   ↓
[12~14층 Stylist] 정서·문단·문장               ⚖️ GATE-12·13·14
   ↓
[Reviewer] 0~14층 자기 검수
   ↓
[출력] 한국어 시나리오 → video_labs 입력
```

각 GATE마다 사용자가 *멈춤·검수·승인·수정·거부* 가능. 자세히: `11_agent_system.md`.

### 14층은 *적는 것이 작품을 쓰는 것*

위→아래로 적어 내려가는 것이 곧 작품의 형태가 펼쳐지는 것. *깊어지면서 동시에 넓어진다* — 거꾸로 된 피라미드. 모든 잎(문장)이 같은 뿌리(전제)에 매달려 있음. 자세히: `02_layers.md`.

## 진짜로 남는 것

작품 자체는 거의 다 사라진다. 진짜로 남는 건 *작업 중 사용자한테 일어난 것* — 판단력·취향·자기 시점의 단단해짐. 그래서 *내가 결정한 것의 누적*이 곧 *나*가 된다. 코덱스가 두꺼워질수록 그 누적이 가속된다.

추가 — 에이전트 시스템에서: *내가 매크로 craft 자체를 하지 않아도* 됨. 그러나 *어느 방향이 옳은지* 판단하는 자리는 사용자에게 남음. 그 판단의 누적이 voice. **에이전트는 voice를 보조하지 대체하지 않는다.**
