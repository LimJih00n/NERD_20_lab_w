# NERD_20 작업 코덱스

AI 협업 시나리오·소설 작업 매뉴얼. 매번 새 작품 시작할 때 옆에 띄워 두고 쓸 작업 인프라.

## 무엇인가

이 코덱스는 *매크로 라이브러리*가 아니라 **상상력 보철물**이다. LLM이 구조적으로 가지지 못한 14차원의 상상을 외부에서 공급하는 시스템. 시간이 갈수록 두꺼워지고, 두꺼워질수록 출력이 *사용자만의 작품*에 가까워진다.

## 어떻게 쓰는가

새 작품 시작 시:
1. `00_identity.md` 의 정체성 3줄 확인
2. `06_writing_order.md` 에서 KNOW → SHAPE → WRITE 순서 따라가기
3. SHAPE 단계에서 `07_beat_sheet.md` 형식으로 비트 시트 작성
4. WRITE 단계에서 `04_macros.md` 의 매크로 5~7개 호출
5. `03_anti_templates.md` 의 금지 항목 적용
6. 작업 후 발견된 새 매크로·새 안티템플릿을 코덱스에 추가

## 진짜로 남는 것

작품 자체는 거의 다 사라진다. 진짜로 남는 건 *작업 중 사용자한테 일어난 것* — 판단력·취향·자기 시점의 단단해짐. 그래서 *내가 결정한 것의 누적* 이 곧 *나*가 된다. 코덱스가 두꺼워질수록 그 누적이 가속된다.
