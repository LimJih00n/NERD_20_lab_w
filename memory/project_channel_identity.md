User is building a content channel. Working hypothesis for the brand color (subject to revision through actual production):

> 동시대를 SF/우화 프레임으로 풍자하는 짧은 영상. 어떤 인물도 결백하지 않고, 미감은 잔혹하다.

**Aesthetic threads:**
- SF / cyberpunk visual register
- Mythological / fable structure (Black Myth: Wukong-style brutal greed parables)
- Contemporary Korean social satire (e.g., SK hynix bonus politics)
- Brutal / visceral aesthetics (Berserk, Park Chan-wook, Cronenberg, I Saw the Devil)
- No moral whitewashing — workers, unions, politicians, public all implicated equally
- Violence / horror register accessible: Game of Thrones / Saw / Narcos level achievable for craft

**Production methodology — see [[collaboration-model]].**

Short version: writer writes a voice-rich treatment per work (quoted dialog, tone instructions, internal thoughts embedded). AI decomposes treatment → 비트 → shot. Writer fills dialog placeholders and does final cut. No shared codex across works.

**Retired hypothesis (2026-05-15 to 2026-05-17):** Earlier working theory framed AI as needing a "14-layer macro library" / "imagination prosthetic" to supply imagination dimensions AI structurally lacks. User tested and dropped this — concluded that per-work voice-rich treatments carry the same load without the maintenance overhead. If treatment-only model breaks (voice-thin treatments, volume scaling, cross-work consistency need), reconsider.

**User's working style observations:**
- Rejects sentence-level "caboose" interpretive postscripts (see [[anti-template-caboose]])
- Rejects label-heavy metaphor
- Rejects option menus — wants one decisive choice
- Rejects forced poetic abstractions ("그것은 ~였다")
- Catches average-quality output quickly
- Engages in long iterative dialogue to crystallize working models
- Will explicitly call out when AI redirects upstream unnecessarily ("멍청하게 굴지마")
