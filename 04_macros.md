# 매크로 라이브러리

## 매크로의 본질 — **AI 상상 공간의 지도**

매크로는 *고립된 도구*가 아니다. **AI의 상상 공간 크기를 *확장 + 깊이* 두 방향으로 키우는 지도**다.

- AI는 *익숙한 패턴*만 생성 → 평균값
- 매크로 = *그 자리의 가능 공간*을 보여주는 지도
- 지도를 받으면 AI가 *익숙하지 않은 자리*도 탐험 가능
- 변주·실패 모드·인접 매크로까지 박혀 있어야 *지도가 작동*

= 매크로 한 줄 정의로는 지도가 안 그려짐. 호출해도 AI가 *평균값 변주*만 함.

## 매크로의 구조 — 7 차원

```
[본질]          무엇을 하는 매크로인가 (한 줄)
[메커니즘 룰]   어떻게 작동하는가 (구체 craft 결정 5~7개)
[시각화 룰]     어떻게 보여주나
[실패 모드]     잘못 적용했을 때 어떻게 망가지나
[인접 매크로]   함께 쓰면 시너지/충돌하는 다른 매크로들
[변주 사례]     다른 작품·다른 장르에서 어떻게 변형되어 나타나나
[재료]          진짜 흔들렸던 작품에서 추출된 구체 사례
```

**상상 공간 결손의 자리들** (각 섹션이 빈 자리 = 상상력 결손):
- 빈 [재료] → 평균값 양산
- 얕은 [메커니즘 룰] → 상상 공간 좁음
- 실패 모드 없음 → AI가 *언제 망가지는지* 모름
- 인접 매크로 없음 → 시너지 못 씀
- 변주 사례 없음 → *변형 가능 공간*을 못 봄

→ 7 섹션 모두 채워야 *지도가 작동*. 빈 섹션 한 칸 = 그만큼 상상력이 좁음.

## 깎음의 작동 — 위층 결정이 매크로를 *자동 호출*

매크로는 사용자가 *수동 호출*하는 도구가 아니라 — **위층 결정이 가리키는 자리**다.

예: 0층 *색채 우선형* + 1·2층 *친밀 무기화 주제* 박히면 — Stylist 에이전트가 자동으로 호출 후보:
- 위선 매크로 (친밀 무기화의 디테일)
- 비애 매크로 (상실 자리)
- 잔혹·학살 매크로 (베다의 자리)
- 복합 정서 매크로 (분리 불가능한 정서)

→ **위층 결정이 매크로를 자동 호출.** 사용자가 *고민하지* 않음. 결정·승인만 함. (자세히: `11_agent_system.md`.)

## 매크로 깎음(sculpting) vs 매크로 발견(discovery)

| | 발견 | 깎음 |
|---|---|---|
| 작업 | 작품 분해 중 *후보 짚음* | 그 후보의 *상상 공간 전체*를 7 차원으로 채움 |
| 결과 | 한 줄 정의 + 한 자리 출처 | 본질+메커니즘+시각화+실패+인접+변주+재료 |
| 시간 | 분 단위 | 30분~1시간/매크로 |

→ 분해 1회 = 후보 N개 발견. 깎음은 별도 사이클. 후보 누적되면 *가장 호출 빈도 높은 것부터* 깎음.

## 깊은 매크로 예시 — *결정 박탈 매크로* (베르세르크에서 발견·깎음)

```
[본질]
주체가 행동할 수 없게 *물리적으로 묶임* → 목격자로 강제 전환.
주체성 박탈된 자리에서 *보는 행위*가 가해의 일부가 됨.

[메커니즘 룰]
- 묶임은 *물리적*이어야 함 (정신적 마비는 별도 매크로)
- 묶는 도구는 *유기적*이거나 *적의 일부* (사도 입·손·촉수)
- 주체가 *발버둥치는 것*을 보여줌 → 무력 강조
- 주체의 *시선만* 강제됨 → 모든 감각 중 *눈*에 무게
- 묶는 행위는 *해치지 않음* — 의도는 가해가 아니라 *보게 만들기*
- 묶임이 풀리는 자리는 *자기 신체 손상*과 함께 (자기 절단 매크로로 자연 이행)

[시각화 룰]
- 묶임의 클로즈 (어디·어떻게 묶였나)
- 발버둥의 작은 연속 컷
- 시선의 클로즈 (눈만)
- 묶임 자체보다 *보는 자리* 강조

[실패 모드]
- 묶임 너무 약함 → 주체가 풀려나 행동 → 매크로 죽음
- 묶음이 *가해*가 됨 (씹음·죽임) → 강제 목격 사라짐, 일반 폭력 매크로로 추락
- 묶음 직후 사건 종료 → 목격할 시간 없음. *느린 의식*이 필요한 매크로.
- 주체 주변에 도움자 등장 → 무력 깨짐

[인접 매크로]
+ 강제 목격 시점 (9층 짝 — 시점 차원)
+ 시선 가두리 (10층 짝 — 둘러싼 머리들이 시선 벽)
+ 시간 느림 (13층 짝 — 묶임 동안 시간 늘어남)
+ 반응 부재 (11층 짝 — 묶인 자가 비명 못 지름)
→ 자기 절단 (8층 — 묶임 푸는 자리에서 자연 이행)

[변주 사례]
- 만화/액션: 사도 입에 손 물림 (Eclipse)
- 일상/심리: 약물 마비 (영화 *Audition*)
- 사회: 군중·권력에 의한 행동 박탈 (체르노빌 *공무원이 진실 못 말함*)
- 영상 변환: 카메라가 *못 움직이는 인물*을 trap. 광각 → 클로즈 패턴.

[재료]
- Eclipse 비트 6-2~6-6 (사도 입에 가츠 손 → 캐스카 사건 → 자기 절단)
- (다른 작품 분해 시 한 단락씩 누적)
```

→ 이렇게 깎아야 에이전트가 호출했을 때 *그 매크로의 상상 공간 전체*를 활용 가능.

작업 시작 시 호출: *"이번 장면에 [매크로명] + [매크로명] 적용."* — 에이전트가 자동으로 7 차원 다 읽고 적용.

## 정서 매크로

### 역겨움 매크로

```
[규칙]
- 시각보다 냄새·온도·점성 우선
- 인물의 신체 반응 1가지 이상 (침 고임, 메스꺼움, 닦이지 않는 잔여)
- 반응 후 그 행동을 계속하는 모순
- "끔찍하다" 같은 저자 해석 0개
- 닦으려는 행동이 더 묻게 만드는 디테일

[재료]
(사용자가 채울 자리 — 검은신화 오공 거북 해체 / Berserk Eclipse /
 Salò 식탁 장면 / 곡성의 어느 자리 등에서 추출)
```

### 위선 매크로

```
[규칙]
- 인물이 도덕적·이타적 언어를 말함 (구체 어휘: "함께", "지키다",
  "민중", "정의", "다음 세대")
- 동시에 인물의 신체가 모순된 행위 중 (잔을 채움, 비싼 시계를 만짐,
  부드러운 천을 만짐)
- 주변에 같은 부류의 인물 1명 이상이 고개를 끄덕임
- 카메라/시점은 거리를 둠. 해석 안 함.
- 마지막 컷: 물질적 디테일 한 가지 클로즈업 (반지·잔 무늬·창밖 풍경)
- 인물 본인은 자기 말을 진심으로 믿음 (의식적 위선 아닌 자기기만)

[재료]
(사용자가 채울 자리)
```

### 비애 매크로

```
[규칙]
- 눈물·먼 곳 응시 0개 (디폴트 슬픔 표현 금지)
- 남겨진 사물의 물리적 디테일 (체온 남은 의자, 옷 냄새, 마시다 만 컵 위치)
- 일상의 사소한 동작이 갑자기 불가능해지는 순간 한 가지
  (양치질, 라면 끓이기, 신발 끈)
- 시간 흐름을 햇빛 옮겨간 자리로 표현 (시계 사용 금지)

[재료]
(사용자가 채울 자리)
```

### 공포 매크로

```
[규칙]
- 비명·도망·괴물·어둠 0개
- 익숙한 것이 0.5cm 어긋난 자리 한 가지 — 인물이 인지하지만 무엇인지
  즉시 못 짚음
- 소리 사이의 침묵 (반응해야 할 자리에 반응 없음)
- 친밀한 사람의 평소와 같은 행동이 미세하게 다름

[재료]
(사용자가 채울 자리)
```

### 분노 매크로

```
[규칙]
- 소리지름·주먹 쥠·얼굴 붉어짐 0개 (디폴트 폭발 금지)
- 시간이 느려지는 감각 (시계 초침, 상대의 한 마디가 또박또박)
- 몸의 미세 반응 (혀가 어금니 누름, 손가락 하나 떨림)
- 과잉한 평정 (대답이 평소보다 부드럽고 친절함)
- 인물이 그 순간 작은 친절한 행동 (잔 채워줌, 문 열어줌)

[재료]
(사용자가 채울 자리)
```

## 폭력 매크로

### 베르세르크식 학살 매크로

```
[규칙]
- 첫 1~2 휘둘림은 세부 묘사 (각 신체의 어느 부위, 어떻게 갈라지는지)
- 그 다음은 "셀 수 없음" 으로 압축 ("이때부터는 셀 수 없었다")
- 죽은 것이 움직임을 멈추지 않음 (입, 손가락, 턱 — 작은 잔여)
- 도구가 사용자를 깎음 (자기 부상, 갑옷 패임)
- 자기 피와 적의 피 구분 안 됨
- 정적 한 컷 다음 미세 움직임 한 가지
- 카타르시스 0 — 끝나면 즉시 다음 무리
- "베다", "썰다" 단어 줄이고 대상의 신체가 어떻게 되는지에 집중

[재료]
(사용자가 채울 자리 — Berserk Conviction / Lost Children arc 등에서)
```

### 나르코스식 살인 매크로

```
[규칙]
- 가해자가 친밀했음을 1컷에 박을 것 (어깨 두드림, 이름 부름, 가족 안부)
- 피해자는 자기가 죽을 줄 모르는 상태로 마지막 행동을 함
  (한 모금 더, 핸드폰 보기, 웃기)
- 죽음 자체는 1~2 문장 (길게 안 끔)
- "피" 단어 0개. 대신 특정한 물질적 사실 1가지
  ("잔이 깨지지 않았다", "신발이 벗겨졌다")
- 결과의 디테일 (청소, 다음 손님, 가족에게 알리는 방식)
- 가해자의 감정 묘사 0개
- 음악적 강조 0개

[재료]
(사용자가 채울 자리)
```

### 왕좌의 게임식 처형 매크로

```
[규칙]
- 공개적·정치적 배경 (군중·증인)
- 가해의 목적이 메시지 전달
- 칼이 단번에 통과하지 않음 (두 번째에)
- 동맥은 두 번 펄럭 후 줄어듦 (만화적 분수 아님)
- 신체의 신경 잔여 반응 (어깨가 두 번 떨림)
- 군중에서 한 사람만 박수 후 따라 안 함
- 마지막 컷: 단상의 검은 자국 + 작은 웅덩이

[재료]
(사용자가 채울 자리 — Ned Stark 처형 등)
```

### 시네버그식 바디 호러 매크로

```
[규칙]
- 인물이 천천히 깨닫는 톤. 비명·공포 0개
- "나의 손인데 나의 손이 아닌 듯한" 인지가 행동에 묻어남
- 호러는 변형 자체가 아니라 인물이 자기 몸과 분리되는 자각에 있음
- 변형의 직접 명명 0개 (무엇인지 안 부름, 보이는 것만 묘사)
- 평균값 카탈로그 금지 (주름·털·모공 없음 같은 디폴트)

[재료]
(사용자가 채울 자리 — The Fly / Tetsuo / Annihilation 어디든)

[주의]
이 매크로는 현재 [재료] 가 비어 있다. 적용 시 평균값 출력될 가능성 높음.
실제 작품 분해 후 채워야 작동.
```

## 직격 vs 암각 (Direct / Off-Frame)

같은 사건을 *직접 보여줌* vs *간접 / 프레임 밖*. 매크로 라이브러리의 별도 축.

### 1. 오프스크린 사운드 매크로
카메라는 다른 곳에 있고, 폭력은 소리로만 전달. 보이는 인물의 *얼굴이 듣고 있는 모습*이 폭력의 강도를 알림.

### 2. 거울/반사 매크로
직접 보지 않고, 깨진 유리·물웅덩이·창문에 비친 모습으로 봄. 부분만 보임.

### 3. 외면하는 시선 매크로
카메라가 *따라가던 인물이 외면하는 순간* 같이 외면함. 폭력은 일어났지만 관객은 보지 못함. 들었음.

### 4. 일상 지속 매크로
폭력 한가운데, 카메라가 *그 자리에서 식사를 계속하는 사람*에게 머묾. 폭력이 배경, 평범함이 전경.

### 5. 시간 차 매크로 (사후만)
사건이 일어났음을 알리는 직후의 한 컷만 — 빈 의자, 떨어진 신발, 누워 있는 잔.

### 6. 부분 매크로 (글림스)
0.5초의 짧은 컷. 한 디테일만 — 손가락 하나, 신발 끝, 입.

### 7. 보고된 매크로 (간접 화법)
일어난 일을 *다른 인물이 말하는 것*으로 전달. 말하는 인물의 망설임·표현 회피가 폭력의 무게를 알림.

## 시점 매크로

### 가해자 시점
첫 칼을 든 자가 화자. *"내가 그랬다"*의 자기 폭로 구조. 가장 신랄.

### 목격자 시점
폭력 안 한 사람이 본 것을 전달. 도덕적 거리.

### 시점 없음 (카메라)
관찰자 없음. 카메라만. 가장 차가움.

### 집단 시점
"우리" 화자. 코러스. 익명성.

---

## 시각 매크로

### 와이드-클로즈-와이드
한 비트의 디폴트 구성. 환경 → 디테일 → 결과.

### 침묵하는 와이드
대사 0개의 풍경 5초 이상. 정적의 무게.

### 부재의 컷
있어야 할 것이 없는 자리. 그 자리에 *없음*을 보여줌으로 강조.

### 뒷모습 추적
인물의 얼굴 안 보임. 등으로 따라감.

### 프레임 밖 사운드
보이지 않는 곳의 소리. 화면 안 인물의 반응이 정보 전달.

## 비트 매크로

### 발견 5단계
1. 설정 (평범한 행동)
2. 감지 (무언가 다르다는 것을 *행동이 멈춤*으로 알림)
3. 확인 (들어 봄, 들여다봄)
4. 시험 (다른 것과 비교)
5. 정적 (알게 됨. 행동 없음. 끝)

### 셋업-페이오프
초반의 작은 디테일이 후반에 큰 의미로 회수됨.

### 인식 비트
인물이 무언가를 깨달음. 화면이 *깨달은 그 사람의 얼굴*에 머묾.

## 매크로 추가 방법

### 발견 단계 (작품 분해 중)
1. 작품 한 편의 14층 분해 진행 (`references/<work>.md`로 저장)
2. 각 층에서 *기존 매크로에 안 맞는 패턴*을 짚으면 → 후보 추가
3. 후보 = 한 줄 본질 + 한 자리 출처

### 깎음 단계 (별도 사이클)
1. 후보 매크로 한 개 선택
2. 7 차원 채움: 본질 → 메커니즘 룰 → 시각화 룰 → 실패 모드 → 인접 매크로 → 변주 사례 → 재료
3. 다른 작품 분해 시 변주 사례 한 단락씩 누적
4. 한 매크로 깎음 = 30분~1시간

→ 발견은 빠르게, 깎음은 천천히. 한 달에 깎음 5~10개가 현실적.

## 시스템 검증 — *작가가 안티템플릿을 직관으로 지킴*

베르세르크 분해 중 발견: **미우라가 `03_anti_templates.md`의 안티템플릿을 자연스럽게 지킴.**

- 캡션 *드묾* → exposition dump 회피
- 비유 안전망 0
- 형용사 거의 0
- 단문 평균 짧음
- 보여줌 우선

→ 시스템이 *실제 작동하는 작가들의 무의식적 룰*과 일치한다는 증거. 시스템 방향이 옳다는 신호. 다른 작품(봉준호·박찬욱·검은신화 오공) 분해 시 같은 검증 반복 가능.
